# FE DriveShare - StaffAdminWeb

This is a minimal React + Vite starter project created for the StaffAdminWeb front-end.

Recommended Node.js: 16+ (LTS) or newer.

Quick start (PowerShell):

```powershell
# 1. vào thư mục project
cd d:\Big_Project\FE-DriveShare-StaffAdminWeb

# 2. cài dependencies
npm install

# 3. chạy dev server
npm run dev

# 4. build production
npm run build

# 5. xem build local
npm run preview
```

Nếu muốn, tôi có thể giúp bạn chạy `npm install` và khởi động dev server ngay bây giờ (yêu cầu Node.js và npm phải cài trên máy của bạn).
